# EcoCatch
Hackathon Hacking Industry Camp 2017

Le projet EcoCatch a pour but de sensibiliser les ados et cela de manière collective et collaborative.
ECO catch c’est un jeu en realité augmenté pour les ados, qui aide les parents à faire des économies.
